### Operating Model

Ensign operates through a **cluster model** with regional subsidiaries:
- **Bandera Healthcare LLC** (Arizona)
- **Endura Healthcare LLC** (Colorado)
- **Gateway Healthcare LLC** (Multi-state)
- **Korsin Healthcare LLC** (Alaska - recent acquisition)

Each subsidiary operates semi-autonomously with local leadership while leveraging corporate support services.

### Related Entities

**Standard Bearer Healthcare REIT** - Ensign's captive REIT that acquires real estate for portfolio facilities.

**The Pennant Group (PNTG)** - Spun off from Ensign in 2019, focuses on home health, hospice, and senior living.

## Query Used (Atlas)

```sql
-- Trace principal network
SELECT
    c.company_name,
    COUNT(DISTINCT pm.id) AS facilities,
    COUNT(DISTINCT pm.state) AS states
FROM principals p
JOIN principal_company_relationships pcr ON p.id = pcr.principal_id
JOIN companies c ON pcr.company_id = c.id
JOIN entities e ON c.id = e.company_id
JOIN property_entity_relationships per ON e.id = per.entity_id
JOIN property_master pm ON per.property_master_id = pm.id
WHERE p.full_name = 'SOON BURNAM'
GROUP BY c.id, c.company_name
ORDER BY facilities DESC;
```
